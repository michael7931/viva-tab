# viva tab 浏览器扩展

这是 `viva tab` 的 Manifest V3 扩展部分，负责弹窗、归档管理页面、设置和右键菜单。

它需要配合父目录中的 [JSMod](../vws-jsmod) 才能读取和操作 Vivaldi 工作区。

## 加载方式

1. 打开 `vivaldi://extensions`。
2. 启用「开发者模式」。
3. 将本目录拖入扩展页，或选择「加载已解压的扩展程序」后选中本目录。
4. 安装并重启 JSMod 后，点击扩展图标确认状态显示“已连接 JSMod”。

完整安装、更新和限制说明见项目根目录的 [README](../README.md)。
